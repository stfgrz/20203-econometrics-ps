# 20203 Econometrics Problem Set 1

This folder contains the solution and resources for Problem Set 1 (Assignment I).

## Repository Structure

- **`data/`**: Contains the datasets used for the assignment (e.g., `assignment_data_group_26.dta`).
- **`src/`**: Contains the Stata do-files (e.g., `20203_ps1_SOL_g26.do`) used to perform the analysis.
- **`output/`**: Stores files generated by the analysis, such as tables (`tables.tex`, `tables.log`) and graphs (`.gph` and `.pdf` files).
- **`docs/`**: Includes the original assignment instructions (`Instructions Assignment I.pdf`) as well as a markdown conversion (`Instructions_Assignment_I.md`).

## Usage

1. Open `src/20203_ps1_SOL_g26.do` in Stata.
2. The code will set its working directory to the `ps1` folder or a relevant path according to the user. It loads the dataset from the `data/` directory.
3. Tables and logs will be output automatically to the `output/` directory. (Note: Some graphs, like `histogram q1.gph`, might need manual export via `graph export` if you wish to recreate them as PDFs/PNGs).
